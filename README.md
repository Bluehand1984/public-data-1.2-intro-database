## Lesson Objectives

Learners will understand:

- Relational database
- Conceptual, logical and physical data model
- Normalization
- Data modeling
- Schema design
- Entity-relationship diagram (ERD)

Learners will be able to:

- Design database schema using DBML
- Create ERD from case study
